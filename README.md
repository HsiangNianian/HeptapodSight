# 《七肢桶之视》  
## *Heptapod Sight* — A TRPG of Foreknown Love  

> **“目的与方法，本是一体。”**  
> 一套实验性桌面角色扮演游戏规则，探索：**当你已看见自己的一生，是否仍愿开始？**  
> 灵感源自 Ted Chiang 《你一生的故事》，但不止于“降临”——这是关于**母亲、离别、原谅，与明知结局仍赴约的勇气**。

这不是传统 TRPG——  
- 无属性、无技能表、无战斗轮  
- 仅需 **3 个问题构建角色** + **3 枚骰子** + **一支笔**  
- 采用 **双轨叙事机制**：线性行动 + 非线性记忆闪回  
- 语言即仪式：写下“我已看见……”可改写局部现实  

> **适合**：1–3 人短团｜文学爱好者｜想尝试非战斗 TRPG 的新人｜哲学系深夜团

## 快速开始

### 阅读核心规则  
[**core-rules.md**](core-rule.md) — 完整规则（约 4 页打印版），新手建议：重点看 **「织命三问」** + **「目的骰流程」** + **「初语系统」**

### 准备材料
| 物品 | 说明 |
|------|------|
| 纸与笔 | 每人至少 1 张（**手写是仪式的一部分**） |
| 3 枚 d6 | 或用 [Roll20](https://roll20.net/) / Discord `/roll 3d6` |
| 安静空间 | 建议开场前静默 1 分钟 |

### 开一场 60 分钟微团
1. 全体闭眼，回想：*“哪一刻让你愿重来一遍？”*  
2. 每人说一个词（不解释）→ 成为本场「世界基调」  
3. 回答 **[织命三问](core-rules.md#-角色创建织命三问)** 创建角色  
4. 引导者公布 **终语之刻**（例：“一年后，你在她空房间，窗台有半块融化的草莓糖”）  
5. 从 **初遇之日** 开始，让记忆自然浮现……

> tips：第一次可仅用 **线性轨 + 环形轨**，暂不检定；专注感受“已知结局下的选择”。
